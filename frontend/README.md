## Getting Started

### Prerequisites

- Node.js & npm ([Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation

```sh
git clone git@github.com:himasnhu018/Odoo.git
cd SkillSwap
npm i
```

### Development

Start the development server:

```sh
npm run dev
```

### Editing Code
